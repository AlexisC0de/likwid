registerRegion()

Register a code region in the Marker API. This is an optional function to remove the register initialization overhead at startRegion. If you don't call this function, startRegion will do the registering.

Direction Data type(s)
Input Parameter
regionTag Identifier for the region.
Returns
errorCode 0 for success, -EFAULT if LIKWID Marker API is not properly initialized.
